# 🌿 Call of Nature: Vanilla Chrome Web Game

Welcome to **Call of Nature**, a 2D environmental adventure game! 

---

## 🎮 THE SIMPLEST WAY TO PLAY (INSTANT 1-CLICK LAUNCH)

If you downloaded or exported this project from AI Studio (via ZIP or GitHub), **you do NOT need Node.js, npm, or terminal commands.**

We have created two standalone offline HTML5 game files:
1. **`Vanilla_Call_of_Nature.html`** (Pure Vanilla HTML5/CSS/JS — 100% human-readable source code!)
2. **`Call_of_Nature_Chrome_Game.html`** (Full pre-bundled React 19 edition)

### 👉 CRITICAL WINDOWS RULE (MUST READ):
⚠️ **DID YOU DOWNLOAD A `.ZIP` FILE ON WINDOWS?**
If you double-click `index.html` inside Windows Explorer's ZIP preview without extracting it, clicking links will say *File Not Found*. 
**Solution:** Right-click the downloaded `.zip` file -> Select **Extract All (Unzip)** -> Open the extracted folder -> Double-click **`Vanilla_Call_of_Nature.html`**! It opens instantly in Google Chrome, Edge, Safari, or Firefox.

## 🛠️ For Developers: Running the Modular Source Code

If you want to edit the React & TypeScript source code (`src/App.tsx`):

### 💻 Option A: One-Click Launcher (Windows / Mac)
- **Windows**: Double-click **`play.bat`** (installs Node.js & dependencies automatically).
- **Mac/Linux**: Double-click **`play.command`**.

### ⌨️ Option B: Command Line (Terminal)
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start local dev server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.
4. To rebuild the standalone offline HTML file after making code changes:
   ```bash
   npm run build
   ```
